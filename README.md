# Retail Customer Behavior & Shopping Trends Analysis

## Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, and Power BI to analyze retail customer behavior and shopping trends. The objective is to transform raw transactional data into actionable business insights that support data-driven decision-making.

## Objectives

* Clean and prepare retail transaction data for analysis.
* Explore customer purchasing patterns and demographics.
* Perform business analysis using SQL queries.
* Build an interactive Power BI dashboard.
* Generate insights and recommendations for business growth.

## Tech Stack

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* MySQL
* Power BI
* Jupyter Notebook

## Project Workflow

### 1. Data Preparation & EDA (Python)

* Data cleaning and preprocessing
* Handling missing values and duplicates
* Exploratory Data Analysis (EDA)
* Feature understanding and trend analysis

### 2. Data Analysis (SQL)

* Revenue analysis
* Customer segmentation
* Product category performance
* Purchase behavior analysis
* Business KPI generation

### 3. Dashboard Development (Power BI)

* KPI tracking
* Customer and sales insights
* Interactive filters and visualizations
* Executive-level reporting dashboard

## Key Insights

* Identified high-value customer segments.
* Analyzed spending patterns across demographics.
* Evaluated top-performing product categories.
* Discovered seasonal purchasing trends.
* Highlighted opportunities for customer retention and revenue growth.
